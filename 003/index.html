QQ悄悄话破解

发表于 2017-09-26 | 分类于 破解

前言

关于悄悄话的破解的讨论，我首先在QQ贴吧发了帖(已被删除)，后来将详细的破解教程放在了我的知乎回答里。现在贴上知乎链接：QQ悄悄话破解-石伟同學的回答
现在将知乎内容搬运到我的博客，下面是教程。

准备工具

  • 版本为5.1.0的qq安卓版客户端（旧版本不好找，一些应用商店的历史版本里面可以找，如pp助手）。
  • 已root的安卓版手机一枚。
  • 安卓版数据库编辑器（百度搜网上有很多）。

操作

  1. qq登陆，并且消息列表有匿名者发你的消息（悄悄话信息不要删除）
  2. 打开数据库编辑器，给予其root权限。首先进入date/date/com.tencent.mobileqq/databases，这一部分的目录是固定的，再打开数据库你的qq.db。 你qq若是12345，数据库名字便是12345.db。
  3. 打开表SecMsgSession.里面每一行数据对应着一组qq匿名对话。找到你要找的一行数据（如果有多条，把其他的匿名对话给删，剩下的就是你要找的），更改属性值mySuid为1（1表示我向其他人发匿名，2表示他人向我发匿名），更改后保存数据并退出编辑器软件。
  4. 进入qq，在消息列表中点击QQ悄悄话并进入（如果你看了视频会发现有个退出qq的操作，其实这一步可有可无，因为从消息列表点击悄悄话进去会有加载，这一步已经有数据的刷新）。
  5. 你会发现对方的头像和昵称会显示出来，自己头像变成匿名者。权利反转的赶脚，不过这是假的权利反转。此时你们都知道对方是谁。

提醒

  • 你完全按照要求做并不能保证一定会找到对方。贴吧的跟帖中，只有一人表示可以。很多人说qq会闪退，不会闪退的也有很多人说对方信息无法显示出来。
  • 尽量用5.1.0 版本，用高的版本很可能闪退。
  • 数据库打开文件有人说是空的，说明你没有root或者编辑器没有赋予root权限。

总结

总结网上的一些方法，发现不是很可行。

  • 用re文件管理器找pre开头文件，我发现根本找不到这个文件。
  • 将自己空间地址发给对方，对方大意没准点了呢。实际中发现无法识别链接，点不开。
  • 登陆webqq,我发现还是不可行。
  • 诈对方说自己已经晓得对方了，并结合说话风格识别。看运气，有可能找到对方。

我自己尝试了从三个不同方法找到对方，上方教程说的是第一个方法。方法二需要一些套路和运气并且需要一些技术，只要套路够有很大几率会得到对方的手机型号和手机号码。第三个方法是个半残品。我发现了qq号和本地数据命名的一些规律，但只是冰山一角，还是无法得出对方。后2条方法最近会更。
