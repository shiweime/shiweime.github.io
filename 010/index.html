屏幕ocr识别软件

发表于 2017-10-28 | 分类于 编程

前言

之前有份工作需要将屏幕上的信息录入系统，我便想到了用软件识别再将识别结果copy到粘贴版。在网上搜索了一下，pdf转word的这类我就不说了，不符合要求。后来也找到了几个比较符合要求的软件。第一个名字叫啥coco文字识别，好几年前的破解版。也许是太久了，识别速度极慢，平均等待近一分钟，识别率也差，也没有复制粘贴版功能，直接排除。第二个叫夕风识别软件。较第一个比较好，用的是百度云ocr接口，识别率比较高，但是缺点也是有的，识别内容不会自动复制到粘贴版，每次识别还得要点击一下截图按钮才能识图，也被我抛弃了。于是我就自己做一个吧。想快速地做一个，听说易语言比较容易上手，于是就用易语言来写了。1.0版前前后后花了2天，真正用在编程上的时间大概是10个小时。这星期又花了大半个星期来优化，自己感觉还蛮好使的。

注意事项

软件是由易语言写的可能会报读。win10自带的Windows Defender仅在1.2版软件写配置文件时候会报毒。室友的装有360的电脑在点击跳转我的博客时候会拦截。可将软件添加信任，介意勿用。

软件下载

意截图百度云下载——密码:uwec

Api Key&Secret Key

注册百度云并实名认证，创建应用，将参数Api Key&Secret Key配置到软件里。软件里面默认参数是我的，有免费额度限制，也不支持并发(多人同时使用没有参数返回)长期使用的话建议按照上面流程去申请自己的接口。
