删除百度失效索引

发表于 2017-11-22 | 分类于 网站

前言

这个博客域名在我注册前有人用过，写过几个博客，百度搜索会有失效链接。这篇文章记录处理过程。

操作

  1. 找出失效索引
    在百度搜索框中输入:site:blog.shiwei.me。

  2. 用百度站长工具处理
    点击此处并登陆百度站长平台
    进入：网站资源>>死链提交

  3. 制作死链文件
    我将文件保存为die.txt

  4. 文件上传至我的网站根目录
    上传至网站根目录

  5. 提交死链文件地址
    提交成功等待处理。

遇到的坑

  1. 编码方式
    编码方式一定要为UTF-8，不然会像下图这样。

  2. 提交失效链接文本地址非失效链接地址

  3. 提交失效链接文件至本网站根目录
    提交至其他地址会出现这个情况

  4. 规则提交无法处理，不解。

总结

Google处理时间未知，等删除好了再在此处补充。
后来补充：Google爸爸处理的真快，昨天写提交的今天就给解决了。
